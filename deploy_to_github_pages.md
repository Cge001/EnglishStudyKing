# GitHub Pages 部署指南

## 步骤1：创建GitHub仓库
1. 在GitHub上创建新仓库，命名为 `testapp-intro`
2. 将以下文件上传到仓库：
   - `app_intro.html` (重命名为 `index.html`)
   - `TestApp-debug.apk`

## 步骤2：启用GitHub Pages
1. 进入仓库设置 (Settings)
2. 找到 "Pages" 选项
3. 选择 "Deploy from a branch"
4. 选择 "main" 分支和 "/ (root)" 文件夹
5. 点击 "Save"

## 步骤3：获取公网地址
部署完成后，你会得到一个类似这样的地址：
`https://your-username.github.io/testapp-intro/`

## 步骤4：重新生成二维码
使用新的公网地址重新生成二维码，这样全世界的人都能访问了！
