# Vercel 部署指南

## 步骤1：准备文件
创建一个文件夹，包含：
- `index.html` (重命名 app_intro.html)
- `TestApp-debug.apk`

## 步骤2：部署到Vercel
1. 访问 https://vercel.com
2. 注册/登录账号
3. 点击 "New Project"
4. 选择 "Upload" 或连接Git仓库
5. 上传你的文件夹
6. 等待部署完成

## 步骤3：获取地址
Vercel会给你一个地址，类似：
`https://testapp-intro.vercel.app`
